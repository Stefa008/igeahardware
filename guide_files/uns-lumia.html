Windows 10 Mobile su Lumia non supportati



In questa guida vi spiegheremo come installare Windows 10 Mobile

su Dispositivi Nokia/Microsoft Lumia non supportati.



PREREQUISITI



  • ⚫ Un Computer Windows

  • ⚫ Un Lumia funzionante [Trasferimento file funzionante
    tutti i tasti funzionanti (Volume + e -, Accensione)]

  • ⚫ Batteria del dispositivo mobile carica
    almeno al 70%

  • ⚫ Windows Phone Internals (Scaricabile da quì).

  • ⚫ Zip contenente il file di aggiornamento alla prima build di W10M
    (Scaricalo da quì.)

  • ⚫ iutool (contenuto all'interno del file scaricato precedentemente ⬆️ ⬆️)
  • ⚫ Lumia aggiornato a Windows Phone 8.1


Passo 1 (facoltativo)



Per prima cosa procediamo a sbloccare
il bootloader del nostro dispositivo.
Per farlo collegate il vostro lumia al pc, aprite WPinternals
e andate nella sezione "Unlock Bootloader"



Cliccate su ok per riavviare il dispositivo in Flash-mode


Dopo che il dispositivo si sarà riavviato in Flash-mode
per sbloccare dovrede scaricare alcuni file che troverete quì sotto.

  • ⚫Engineering SBL3 Download

  • ⚫Emergency Files Download

  • ⚫File di aggiornamento .ffu (Varia per Ogni dispositivo,
    assicuratevi di scaricare la versione corretta) Download

  • ⚫Infine seguite le istruzioni a schermo per completare lo sblocco



Passo 2

Installazione di Windows 10 Mobile



  • ⚫Caricare il dispositivo almeno fino all'80%

  • ⚫Collegarlo al PC con un cavo che permetta anche il trasferimento dati

  • ⚫Scaricare e scompattare tramite 7zip (Scaricalo quì.) il file contenente i file di aggiornamento da quì.

  • ⚫Dopo aver scopattato il file scaricato creare una cartella sul desktop con nome a piacere
    e copiare la cartella contenuta nel file .wim di aggiornamento scaricato precedentemente corrispondendte
    alla risoluzione del proprio dispositivo. Per scoprirlo basta andare nelle impostazioni nella sezione info sul dispositivo.



  • ⚫Ora andate nella cartella dove si trova iutool e aprite all'interno di quest'ultima una finestra del cmd.
